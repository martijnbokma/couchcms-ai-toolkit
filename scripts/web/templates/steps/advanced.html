AAPF Framework

What is the AAPF Framework?

The AAPF (Autonomous Agent Programming Framework) framework provides structured and predictable behavior for AI agents. It includes doctrine (core principles), directives (communication guidelines), playbooks (workflow templates), and enhancements (advanced features).

Enable the AAPF framework for more structured and predictable AI agent behavior. This is especially useful for complex projects where consistency is important.

Context Directory

What are AI context files?

AI context files contain project-specific information that AI agents use to better understand how your project works. This can include coding standards, architecture documentation, or custom patterns that agents need to know.

Specify a custom directory where AI context files will be stored. By default, .project/ai is used. You can change this if you prefer a different location or want to use multiple context directories.
