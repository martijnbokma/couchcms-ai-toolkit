AAPF Framework

Wat is het AAPF Framework?

Het AAPF (Autonomous Agent Programming Framework) framework zorgt voor gestructureerd en voorspelbaar gedrag van AI agents. Het bevat doctrine (kernprincipes), directives (communicatie richtlijnen), playbooks (workflow templates) en enhancements (geavanceerde features).

Schakel het AAPF framework in voor meer gestructureerd en voorspelbaar gedrag van AI agents. Dit is vooral nuttig voor complexe projecten waar consistentie belangrijk is.

Context Directory

Wat zijn AI context bestanden?

AI context bestanden bevatten project-specifieke informatie die AI agents gebruiken om beter te begrijpen hoe jouw project werkt. Dit kunnen coding standards, architectuur documentatie, of custom patterns zijn die agents moeten kennen.

Specificeer een aangepaste directory waar AI context bestanden worden opgeslagen. Standaard wordt .project/ai gebruikt. Je kunt dit aanpassen als je een andere locatie prefereert of als je meerdere context directories wilt gebruiken.
