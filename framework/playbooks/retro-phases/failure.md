# Phase 2: Failure Analysis

---

## Failure Patterns

**For each failure:**

```markdown
## Failure: {Name}

**What went wrong:**
{Description of the failure}

**User correction:**
{How the issue was identified}

{Why it happened - be specific}

{What should be done differently}
```
